# KS Boil Water Curriculum

This repository hosts a Jupyter Book site and supporting notebooks for the KS Boil Water Curriculum.

## Quick start

```bash
pip install -r requirements.txt
jupyter-book build book
```

The built site will be in `book/_build/html`.

## Colab notebooks

Update the GitHub org/repo placeholders in `book/_config.yml`, `book/intro.md`, and `book/tutorials/day1_intro.md` so the Colab links point to your repository.
